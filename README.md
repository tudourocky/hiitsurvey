## Version Management

⚠️ **Important**: This project uses specific Python and Node.js versions. Before starting development, please read [VERSION_MANAGEMENT.md](VERSION_MANAGEMENT.md) to ensure you're using the correct versions.

**Required Versions:**
- Python: 3.14.2
- Node.js: 25.2.1

The project includes `.python-version` and `.nvmrc` files to help automatically set these versions if you use `pyenv` or `nvm`.
